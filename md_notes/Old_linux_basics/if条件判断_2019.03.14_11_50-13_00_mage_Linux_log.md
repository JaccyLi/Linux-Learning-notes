**条件判断，控制结构：**

###### 单分支的 if 语句：
if 判断条件; then
		statement1
		statement2
		...
	fi
	
###### 双分支的 if 语句：
if 判断条件; then
		statement1
		statement2
		...
	else
		statement1
		statement2
		...
	fi

###### 多分支的if语句：
if 判断条件1； then 
	statement1
	...
	elif 判断条件2；then 
	statement2
	...
	elif 判断条件3；then
	statement3
	...
	else
	statement4
	...
	fi
	
**练习:写一个脚本，完成以下要求**

给定一个用户：
		1，如果其UID为0，就显示此用户为管理员
		2，否则，就先是其为普通用户
```shell
 	#！/bin/bash
	NAME=user6
	USERID=`id -u $NAME`
	if [ $USERID -eq 0 ] ; then
		echo "Admin"
	else
		echo "Common user."
	fi
```
或者
```shell
	#！/bin/bash
	NAME=user6
	#USERID=`id -u $NAME`
	if [ `id -u $NAME` -eq 0 ] ; then
		echo "Admin"
	else
		echo "Common user."
	fi
```
