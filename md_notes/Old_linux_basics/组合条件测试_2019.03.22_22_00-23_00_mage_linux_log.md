###### 组合测试条件
	-a 与关系
	-o 或关系
	！ 非关系
	eg：if [ $# -gt 1 -a $# -le 3 ]
	    if [ $# -gt 1 ] && [ $# -le 3 ]

###### 练习
写一个脚本，接受参数：q,Q,quit,Quit。是这几个参数就退出，否则提示无法识别的参数。
```shell
#!/bin/bash
#
if [ $1 =\= 'q' -o $1 =\= 'Q' --o $1 =\= 'quit' -o $1 =\= 'Quit' ]; then
echo "Quiting..."
else 
echo "Unknown Argumen."
exit 1
fi
```
扩展：如果用户所给==不是==以上4个参数，则创建该参数命名的用户。
